---
title: The PMO as Code portfolio goes public
date: "2026-07-03"
description: "The portfolio that governs PMO as Code is now run as PMO as Code, in public: five projects, gated documents, and a live derived dashboard."
---

The standard now governs its own development in the open. The
[pmo-as-code-pmo repository](https://github.com/c4g-john/pmo-as-code-pmo)
holds five chartered projects covering the standard, the engine, the launch,
and operations, every document gated by the same checks the standard
prescribes, with status derived onto a
[public dashboard](https://c4g-john.github.io/pmo-as-code-pmo/).

On day one, the AI advisory layer walked all 84 traceability links in the
document set and flagged eight findings. All eight held up under review and
were fixed through gated pull requests, including two business requirements
the grader forced into existence because requirements were tracing to
reasoning that had never been written down.

The claim this arrangement backs is simple to check: whatever color the
checks derive is the color the public dashboard shows. The
[self-governance page](/self-governance/) keeps every part of it one click
away.
